# Expense Tracker Web Application
Developed by Rakesh Sharma

A Dockerized Spring Boot + MySQL Expense Tracker application.

## Tech Stack

- Java 17
- Spring Boot 3
- Spring Security
- Spring Data JPA
- MySQL 8
- Docker
- Docker Compose
- Maven

## Features

- User Authentication
- Add Expenses
- Update Expenses
- Delete Expenses
- Expense Categories
- MySQL Database
- Dockerized Deployment

## Project Structure

src/
Dockerfile
docker-compose.yml

## Run

docker compose up --build

Application

http://localhost:8080

Database

MySQL 8
